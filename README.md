1. 第 1 章 - Agent 概念与架构模式
2. 第 11-13 章 - RAG 完整技术体系
3. 第 16 章 - 安全与隐私
4. 第 17 章 - 测试与评估
5. 第 23-25 章 - 工程实践与最新技术

#### 🎯 全面掌握 (22-29 小时)
按顺序阅读第 1-25 章
